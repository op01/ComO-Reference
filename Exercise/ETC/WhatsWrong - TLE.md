What's Wrong - Time limit exceeded
====================
Tags : `Math`<br>
Difficulty : &#9733;&#9733;&#9733;&#9733;&#9733;&#9734;&#9734;&#9734;&#9734;&#9734;<br>
Estimated time solve : 15 min<br>

- - -

**What's Wrong** : โจทย์ชุดนี้จะรวบรวมตัวอย่างโจทย์ยากๆที่คนพึ่งเริ่มอาจจะยังไม่เคยรู้ อาจจะเป็นโจทย์ที่วิธีปกติใช้ไม่ได้ หรือผิดด้วยสาเหตุต่างๆ และแสดงให้เห็นว่าผิดเพราะอะไร

**Time limit exceeded** เป็นคำที่คนแข่งเขียนโปรแกรมทุกคนกลัวมากที่สุด และเกลียดมันมากที่สุด มันหมายความว่าโปรแกรมของเราใช้เวลาเกินกำหนดเช่น 1 วินาที หรืออาจใช้เวลานานมากๆเช่นเป็นวัน เป็นปี (การ brute-force password ต่างๆ อาจใช้เวลาเป็นเดือนๆทั้งที่ใช้คอมพิวเตอร์ช่วยกันคิดกว่าพันๆเครื่อง)

คุณสร้างโปรแกรมตัวหนึ่งขึ้นมาที่มีโค้ดดังนี้
```cpp
int testcase;
scanf("%d",&testcase);
for(int testcaseLoop = 0;testcaseLoop < testcase;testcaseLoop++)
{
    int N;
    scanf("%d",&N);
    long long cnt = 0;
    for(long long i = 1;i <= N;i++)
    {
        for(long long j = 1;j <= N;j++)
        {
            for(long long k = 1;k <= N;k++)
            {
                cnt += i*j*k;
                cnt %= 1000000007;
            }
        }
    }
    printf("%lld\n",cnt);
}
```
(โปรแกรมนี้ทำการรับ testcase มาให้ด้วย)

โปรแกรมตัวนี้ทำงานได้ดีมาก แต่เมื่อคุณลองใส่เลขเยอะๆเข้าไป คอมพิวเตอร์ต้องใช้เวลาคำนวณนานมากๆ (ถ้าใส่ N=1000000 อาจใช้เวลาถึง 30 ปี) คุณจึงต้องเขียนโปรแกรมใหม่ที่ทำงานได้เหมือนโปรแกรมนี้ แต่ใช้เวลาเร็วกว่าเดิม

Input
-----
บรรทัดแรกรับจำนวน testcase **T** (**T** <= 5)

แต่ละ testcase รับจำนวนเต็ม 1 ตัวคือ **N** (1 <= **N** <= 1000000)

Output
------
แต่ละ testcase แสดงจำนวนเต็ม 1 ตัว แสดงถึงผลการทำงานของโปรแกรมข้างบน

Example Input
-------
```
3
1
100
999999
```

Example Output
-------------
```
1
787624104
18503919
```

Note
--------
ลองเอาโค้ดข้างบนก๊อบไป แล้วใส่ Example Input เข้าไปก็ได้ แล้วรอ คุณจะรู้ว่าชาตินี้โปรแกรมก็ไม่มีวันเสร็จ ถึงคุณจะใช้คอมพิวเตอร์ดีแค่ไหนก็ตาม

โจทย์ข้อนี้ใช้ Math ธรรมดา สามารถจัดรูปให้สามารถหาคำตอบได้โดยไม่ใช้ loop แต่ระวังการ overflow ด้วย
